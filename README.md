✈️ Airplane Reservation System
A robust, terminal-based reservation management tool engineered in C. This project demonstrates core systems programming concepts, including 2D array manipulation, memory management through pointers, and modular function architecture.

🚀 Features
Dynamic Seating Chart: Visualizes an 8x4 grid of the aircraft using a coordinate-based mapping system.

Reservation Logic: Allows users to book specific seats (marked as [R]) with built-in checks to prevent double-booking.

Cancellation System: Securely releases reserved seats back to "Available" ([A]) status.

User Profile Management: Captures and displays passenger metadata, including Name, Age, Gender, and Contact details.

Input Validation: Features error-handling for invalid row/column entries and occupied seats to ensure system stability.

🛠️ Technical Stack
Language: C (Standard: C99/C11)

Logic Structures: 2D Arrays, Global Constants (#define), Switch-Case Control Flow, and Boolean flags.

Compiler: GCC / Clang
